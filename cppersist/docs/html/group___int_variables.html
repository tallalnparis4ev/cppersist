template<typename T , typename Ret , typename... Args>
PersistentMemoized< T, Ret, Args... > cpst::getLocalMemoizedObj ( MemCacheType  primaryCache,
string(*)(Args...)  key,
string(*)(Ret)  pickle,
Ret(*)(string)  unpickle 
)

Creates an object, of type T, that has persistent memoization applied to T's solve function. Here, the storage medium for persistent memoization is the local file system.

Template Parameters
T- the class which the user wants to instantiate an object of, with a memoized version of the solve function.
Parameters
primaryCache- this will initialise the specified memory cache and use it as the primary cache.
key- this function should return a string representation for a given argument to the solve function.
pickle- this function should return a string representation for any return of the solve function.
unpickle- this function is the inverse of pickle.
