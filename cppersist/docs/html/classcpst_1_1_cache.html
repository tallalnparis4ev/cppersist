◆ get()

template<typename Ret , typename ... Args>
virtual std::optional<Ret> cpst::Cache< Ret, Args >::get ( const Args &...  args)
pure virtual

This function should attempt to fetch from the cache the value corresponding to the passed in arguments. If no such entry exists, then a nullopt should be returned.

Parameters
args- The arguments to be searched from the cache.
Returns
the value contained in the cache, corresponding to the key formed by args (if no such key exists, the nullopt is returned).

Implemented in cpst::MongoDBCache< Ret, Args >, cpst::DiskCache< Ret, Args >, cpst::RegCache< Ret, Args >, cpst::OneCache< Ret, Args >, and cpst::LRUCache< Ret, Args >.
